Sourcecode of the main UI of the application

```bash
# Install dependencies
npm install

# Run the app for main development
npm start

# Development of the Web Components
npm run start-wc

# Build

npm run build

# Unit tests of UI
npm run build:dev:test
npm run test:unit

# Unit tests of Web Components
npm run test:wc

# E2E tests
npm run build:dev:test
npm run test:e2e
```
